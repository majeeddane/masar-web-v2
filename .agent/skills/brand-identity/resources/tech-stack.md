# التكنولوجيا المفضلة وقواعد التنفيذ

عند إنشاء الأكواد أو مكونات واجهة المستخدم لهذه العلامة التجارية، يجب الالتزام الصارم بالاختيارات التالية.

## التقنية الأساسية
* الإطار: React (يفضل TypeScript)
* محرك التنسيق: Tailwind CSS (إلزامي. لا تستخدم CSS عادي أو styled-components إلا إذا طلب صراحة.)
* مكتبة المكونات: shadcn/ui (استخدم هذه العناصر كأساس لجميع المكونات الجديدة)
* الأيقونات: Lucide React

## إرشادات التنفيذ

### 1. استخدام Tailwind
* استخدم فئات المرافق مباشرة في JSX.
* استخدم رموز الألوان المحددة في design-tokens.json (مثال: bg-primary text-primary-foreground بدلاً من القيم الثابتة HEX)
* الوضع الداكن: دعم الوضع الداكن باستخدام dark: في Tailwind.

### 2. أنماط المكونات
* الأزرار: الإجراءات الأساسية يجب أن تستخدم اللون الأساسي الصلب، والثانوية تستخدم نسخ Ghost أو Outline من shadcn/ui.
* النماذج: ضع التسميات دائمًا أعلى الحقول. استخدم المسافات القياسية في Tailwind (`gap-4` بين العناصر).
* التخطيط: استخدم Flexbox وCSS Grid عبر أدوات Tailwind لجميع التخطيطات.

### 3. أنماط ممنوعة
* لا تستخدم jQuery.
* لا تستخدم فئات Bootstrap.
* لا تنشئ ملفات CSS جديدة؛ احتفظ بالأنماط داخل ملفات المكونات باستخدام Tailwind.
